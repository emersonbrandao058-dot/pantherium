# 🌐 Sobre o Projeto

O **Pantherium** é o site institucional da **Atlética de Enfermagem do Centro Universitário UNEF**.

O projeto foi desenvolvido para apresentar a identidade da atlética, sua história, diretoria e momentos importantes vividos pela comunidade Pantherium.

A ideia do design foi criar um visual **dark premium com detalhes em verde neon**, representando:

* 🐆 Força da pantera
* 🐍 Sabedoria da cobra
* 💚 Identidade da enfermagem
